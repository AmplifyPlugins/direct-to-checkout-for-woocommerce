Redirects WooCommerce customers to the check-out page instead of the cart page.

== Description ==
Direct to Checkout for WooCommerce provides an easy way for WooCommerce customers to go directly to the checkout page after adding a product to their cart.

Direct to Checkout for WooCommerce features include:
* Enables customers to be sent direct to checkout after clicking the Add To Cart button on the shop or product pages.
* Change the default Add To Cart button text on the product page.
* Change the default Add To Cart button text on the shop page for simple products only.
* Add the shopping cart to the checkout page. This allows customers to change quantities, remove products, apply coupons, etc.

By eliminating the cart page from the customer\'s journey on your store, you provide the customer with a much more efficient checkout. This leads to higher converting stores and a more pleasant customer experience.

== Installation ==
= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of Direct to Checkout for WooCommerce, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type “Direct to Checkout for WooCommerce,” then click “Search Plugins.” Once you’ve found this plugin,  you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by clicking “Install Now,” and WordPress will take handle the installation for you.

= Manual installation =

The manual installation method requires downloading the Direct to Checkout for WooCommerce plugin and uploading it to your webserver via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
